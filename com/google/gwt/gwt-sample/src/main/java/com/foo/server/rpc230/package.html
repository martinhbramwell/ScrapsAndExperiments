This package contains the classes necessary to replace the GWT RPC in order that the new RPC uses a filter to work Hibernate. Hibernate filter is in the package gov.senasa.embalajemadera.server.rpc. The name is because gov.senasa.embalajemadera.server.rpc230 has been replaced RPC version 2.3.0. This solution was taken from the book Pro Web 2.0 Application Development with GWT by Jeff Dwyer wrote. ISBN-13 (pbk): 978-1-59059-985-3 ISBN-10 (pbk): 1-59059-985-3
